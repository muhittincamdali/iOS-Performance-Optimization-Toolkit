# Changelog

All notable changes to PerformanceKit will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-04

### Added
- Initial release
- Performance measurement with `measure()` and `measureAsync()`
- Memory usage tracking (resident and virtual)
- CPU usage monitoring per thread
- Statistical analysis (mean, median, min, max, std dev, percentiles)
- Scoped profiling with `Profiler` class
- Nested profiling support with hierarchical reports
- Benchmarking with warmup iterations
- Benchmark comparison for A/B testing
- Benchmark suites for grouped testing
- Instruments signpost integration
- Performance report generation
- Async/await support throughout
- Full test coverage
- Comprehensive documentation

### Platforms
- iOS 15.0+
- macOS 13.0+
- tvOS 15.0+
- watchOS 8.0+
